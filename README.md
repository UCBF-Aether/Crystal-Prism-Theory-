# The Crystal Prism

A mechanical theory of everything. The vacuum is a face-centered cubic supersolid descended from the E8 root lattice through an icosahedral quasicrystal. All of physics -- light, gravity, quantum mechanics, particle masses, dark energy, galaxy rotation, consciousness -- emerges as vibrations, strains, and defects in this single medium.

## Files

- `crystal_prism.py` -- The derivation engine. Computes all fundamental constants from four axioms. One import. Zero empirical inputs.
- `crystal_prism_theory.pdf` -- The mechanical theory paper. Explains what the crystal is and how it works. No math required. The script does the math.

## Quick Start

Run the script in any Python 3 environment with numpy and scipy installed:

python crystal_prism.py

The script will derive the speed of light, Planck's constant, Newton's constant, and the fine-structure constant live, comparing each to CODATA values. It will then compute cosmological parameters, galactic dynamics, and print a summary of all confirmed predictions.
